Enable or disable the Gutenberg editor on selected post types.

== Description ==
This plugin allows administrators to enable or disable the Gutenberg editor for specific post types like posts, pages, and custom post types.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/vaibhaw-gutenberg-editor/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings from Settings > Gutenberg Editor.

== Frequently Asked Questions ==
= Can I enable Gutenberg for only posts? =
Yes, you can select which post types to enable Gutenberg for.
